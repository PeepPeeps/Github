CSS: Font Styling

Style Properties

Property Name Description
<color> color Changes the color of the font.
<font-family> font family Changes the font type.
<font-size> font size Changes the size of the font.

Style Attributes

Attribute Name Description
p{ } paragraph Selector that selects the paragraphs to apply the style to.
.class{ } class Selects all items with same class attribute to apply style to.
#ID { } ID Selects all items with same ID attribute to apply style to.
